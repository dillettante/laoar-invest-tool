# 라오어 투자 도구 (Laoar Invest Tool)

라오어式 **3가지 투자법**(무한매수 · VR 적립식/거치식/인출식)의 **오늘 걸 주문·신호를 자동 계산**해주는 웹 도구.
종목 티커만 넣으면 시세를 불러와 **LOC 매수/매도 사다리**와 홀드/매수/매도 신호를 보여줍니다. 달러·원화 같이 표시.

> ⚠️ **자문·계산 보조용입니다. 투자 권유가 아니며, 주문·체결은 본인이 직접 합니다.** 아래 [면책](#️-면책--주의) 꼭 읽어주세요.

## 이게 뭐예요?

- **3가지 전략**을 한 곳에서: 무한매수(v2.2) · VR 적립식 · VR 거치식 · VR 인출식.
- **종목·차수별로 계좌 여러 개**: 예) `SOXL 무매 1차`, `TQQQ VR 적립` 을 각각 관리.
- **티커 자동 시세**(TQQQ, SOXL 등 아무 미국 티커) + **달러/원화 병기**.
- 매일 **오늘 걸 LOC 주문**과 신호를 계산 → 그대로 증권사에 예약주문 걸면 됩니다.

## 🖥 내 컴퓨터에서 직접 돌리기 (선택)

**Python만 있으면 됩니다.** 설치할 패키지가 하나도 없어요(표준 라이브러리만 사용).

### 1) 파일 내려받기
- 이 페이지 위쪽 초록색 **`< > Code`** 버튼 → **Download ZIP** → 압축 풀기
- (Git 쓰는 분): `git clone https://github.com/dillettante/laoar-invest-tool.git`

### 2) 실행
압축 푼 폴더에서 **터미널**(Mac=터미널 앱, Windows=명령 프롬프트)을 열고:

```bash
python3 server.py
```
> Windows에서 `python3`가 안 되면 `python server.py` 로 하세요.
> Python이 없다면 [python.org](https://www.python.org/downloads/) 에서 3.9 이상 설치.

### 3) 브라우저에서 열기
**http://localhost:8770**

끝입니다. 끄려면 터미널에서 **Ctrl + C**.
같은 집 와이파이의 폰에서 보려면 실행한 컴퓨터의 IP로 `http://<컴퓨터IP>:8770`.

## 📖 사용법

1. **`+ 새 계좌`** — 이름(예: `내 TQQQ VR`), 전략, 티커(`TQQQ`), 그리고 **지금 내 상태**(보유수량·평단·POOL 현금)를 입력하고 저장.
2. 저장하면 티커로 시세를 자동으로 불러와 **오늘 신호(🟢홀드/🔵매수/🔴매도) + LOC 사다리**(무한매수는 오늘의 매수/매도 주문)를 보여줍니다.
3. 실제로 주문·체결했으면 아래 **`체결 반영`**에 수량·단가를 넣어 기록 → 보유·평단·POOL이 갱신됩니다.
4. 상단 **`내보내기`**로 가끔 백업하세요. (데이터가 브라우저에만 있어서, 브라우저 캐시를 지우면 사라집니다. **`불러오기`**로 복구.)
5. 처음 보이는 **예시 계좌**는 가짜 값이니 삭제해도 됩니다.

**입력값이 헷갈릴 때:**
- **평단** = 내 평균 매입가, **POOL** = 아직 안 산 대기 현금, **V(목표)** = VR의 목표 평가금(비우면 보유×현재가로 자동).
- 무한매수 **원금** = 이 종목에 넣기로 한 총액, **분할수** = 40(v2.2)/20(v3·v4), **누적매수액** = 지금까지 실제 산 금액.

## ⚠️ 면책 / 주의

- 이 도구는 **정보 제공·계산 보조용**이며 **투자 자문·권유가 아닙니다.** 모든 판단과 결과 책임은 사용자 본인에게 있습니다.
- **주문·체결·환전은 사용자가 직접** 합니다. 이 도구는 자동으로 주문을 넣지 않습니다.
- 라오어 및 공식 서비스(카페·팬딩 등)와 **아무 관련이 없습니다.** 전략 규칙은 공개된 2차 자료를 교차검증해 구현한 것으로 공식과 다를 수 있습니다.
- **무한매수 v2.2 일부 수치(별지점 산식 등)는 출처마다 상충**합니다(아래). **실제 운용 전 카페 원문으로 확정**하세요.
- **3배 레버리지 ETF**(TQQQ·SOXL 등)는 **원금 손실 위험이 매우 큽니다.**
- 데이터는 브라우저에만 저장 → **정기 백업(내보내기) 필수.**

## 전략 파라미터 (무한매수 v2.2)

기본값은 커뮤니티 **주류값**이며 화면 고급설정에서 바꿀 수 있습니다. 상충/불확실 지점:

- **별지점(큰수) 산식** = `A − B·T` = 기본 **15 − 1.5·T** [주류]. 일부 출처는 `10 − 0.5·T`. → `A/B` 조정 가능.
- **익절률**: TQQQ **15%** / SOXL **20%** / 기타 10% (자동 세팅).
- **쿼터매도**(매일 보유 1/4)·**쿼터손절**(40회 소진·미익절 시 보유 1/4 MOC)은 구현돼 있으나 버전(v2.2/v3/v4)마다 세부가 달라 **카페 원문 확인 권장**.
- v3.0(20분할)·v4.0(리버스·소진모드)은 분할수·익절만 프리셋, 나머지 로직 미구현.

VR: 밴드 ±15%/±25%, POOL 바닥(적립 25%·거치 50%·인출 75%).

## 🛠 직접 배포하려면 (개발자용)

순수 stdlib라 어디든 올라갑니다.

- **Render(무료)**: [이 링크](https://render.com/deploy?repo=https://github.com/dillettante/laoar-invest-tool)로 원클릭 → GitHub 로그인 → Apply. `render.yaml` 포함.
- **클라우드 호스트 일반**: 시작 명령 `python3 server.py $PORT 0.0.0.0` (`Procfile` 포함).
- **임시 공개**: 로컬 실행 후 `cloudflared tunnel --url http://localhost:8770` 또는 `ngrok http 8770`.

사용자가 많아지면 `server.py`의 `yahoo_last`에 티커당 캐시(이미 3분 적용)를 늘리세요.

### 파일 구성
| 파일 | 역할 |
|---|---|
| `server.py` | 무상태 서버(시세·환율·계산). 개인 데이터 저장 안 함. |
| `vr_core.py` | VR 엔진(신호·V밴드·LOC 사다리). |
| `im_core.py` | 무한매수 v2.2 엔진. |
| `index.html` | 프론트엔드(멀티 계좌, 브라우저 저장, USD+원화). |
| `Procfile`·`render.yaml`·`requirements.txt` | 배포 설정(의존성 없음). |

### 자체검증

python3 im_core.py    # 무한매수 엔진 assert 테스트
